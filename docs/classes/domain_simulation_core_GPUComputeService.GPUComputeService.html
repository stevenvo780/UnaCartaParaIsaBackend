    Class GPUComputeService

    GPU computation service abstraction using TensorFlow.js.

    Provides optimized batch operations for entity processing. Automatically falls back to CPU if GPU is unavailable or for small batches. Tracks performance statistics for monitoring.

    See

    • NeedsSystem for needs decay batch processing
    • MovementSystem for position update batch processing
