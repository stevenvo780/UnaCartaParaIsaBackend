    Class GPUComputeService

    GPU computation service abstraction using TensorFlow.js.

    Provides optimized batch operations for entity processing. Automatically falls back to CPU if GPU is unavailable or for small batches. Tracks performance statistics for monitoring.

    See

    • NeedsSystem for needs decay batch processing
    • MovementSystem for position update batch processing

    initialize

    • initialize(): Promise<void>

      Initializes TensorFlow.js and detects GPU availability. Must be called before using any GPU operations.

      Returns Promise<void>

      Remarks

      Performs GPU detection by testing tensor operations. Falls back to CPU if GPU is unavailable or initialization fails. This method is idempotent and safe to call multiple times.

      Throws

      Never throws - errors are caught and logged, CPU fallback is used
