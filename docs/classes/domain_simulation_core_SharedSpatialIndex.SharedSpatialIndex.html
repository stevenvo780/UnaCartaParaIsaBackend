    Class SharedSpatialIndex

    Shared spatial index for all systems.

    Provides O(log n) spatial queries using a grid-based spatial hash. Optimized with incremental O(Δn) updates instead of O(n) full rebuilds. Uses object pooling to reduce garbage collection pressure.

    See

    SpatialGrid for the underlying grid implementation
