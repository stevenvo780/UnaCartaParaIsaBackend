    Class EntityIndex

    Centralized service for O(1) entity and agent indexing.

    Uses granular dirty tracking to avoid O(n) full rebuilds. Only rebuilds when there are actual structural changes to the data. Supports incremental updates for better performance.

    See

    SharedSpatialIndex for spatial queries
