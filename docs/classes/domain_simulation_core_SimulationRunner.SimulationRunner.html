    Class SimulationRunner

    Main simulation orchestrator and coordinator.

    Manages the game state, coordinates all simulation systems, handles commands, and provides snapshots for client synchronization. Uses MultiRateScheduler for optimized system updates at different frequencies.

    See

    • MultiRateScheduler for update rate management
    • StateCache for snapshot optimization
    • DeltaEncoder for delta snapshot generation
