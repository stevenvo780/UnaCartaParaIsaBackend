    Class QuestCatalog

    Catálogo estático de todas las misiones (Quests) disponibles en el juego. Actúa como una base de datos en memoria de las definiciones de misiones.

    Contiene:

    • Misiones de historia principal (main_story)
    • Misiones secundarias (side_quest)
    • Misiones diarias (daily)
    • Misiones de exploración (exploration)
